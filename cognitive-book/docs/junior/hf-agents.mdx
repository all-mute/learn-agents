# AI Agents huggingface open [course](https://huggingface.co/learn/agents-course)

## Course

You will need to complete Units 0, 1, 2, 2.1 of [this course](https://huggingface.co/learn/agents-course)

**Units 2.2 and beyond are not required!**

:::info instruction

- the first modules dedicated to LLM can be PARTIALLY skipped if you are familiar with the information in them
    - keep in mind that in modules with a familiar name, there may be useful information somewhere in the middle/end
- use `https://proxy.merkulov.ai` and your LLM API key [(what is this?)](https://llm-keygen.merkulov.ai)
    - access the proxy by initializing models through OpenAI or LiteLLM classes in frameworks (they are available in langchain, smolagents, llamaindex)

:::
