# Desarrollo rápido

Imagínate que eres un arquitecto del futuro, que construye rascacielos con cubos digitales de Lego. Aquí dominaremos las arquitecturas serverless: tu constructor mágico para la construcción instantánea de sistemas de IA. Aprenderás a ejecutar redes neuronales en la nube como un mago que saca palomas de un sombrero (AWS Lambda), a construir API inteligentes en minutos (Firebase) y a crear sistemas escalables que crecen con tus ambiciones. Estas habilidades se convertirán en tu teletransporte personal al mundo del desarrollo industrial de GenAI, donde la infraestructura se gestiona con un par de clics y los recursos son tan infinitos como el universo.

:::tip Consejo de la Startup School de Y Combinator:
"Siempre usa soluciones listas para usar cuando sea posible."
:::

<details>
<summary>Instrucciones para Ask AI</summary>

:::tip instrucción
Dado que estos temas no cambian con el tiempo, lo mejor es estudiarlos con un tutor personal: ChatGPT.

El proceso de estudio debe ser el siguiente:
- Creas un prompt del sistema para ChatGPT ([plantillas](../metalearning#chatgpt-prompts)), donde describes tus antecedentes, preferencias, nivel de detalle de las explicaciones, etc.
- Copias el tema de la lista (triple clic) y le pides a ChatGPT que te explique este tema
- Si deseas profundizar, haz preguntas aclaratorias

Actualmente, esta es la forma más conveniente de aprender los conceptos básicos. **Además de los conceptos, puedes estudiar materiales adicionales en las secciones Oro, Plata y Extra.**
1. Oro: es imprescindible estudiar antes de comunicarse con ChatGPT
2. Ask AI: haz preguntas sobre cada tema desconocido
3. Plata: materiales secundarios
4. Extra: temas avanzados
:::
