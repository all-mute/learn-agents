### Clasificación de la pregunta por tema

**Prompt del sistema**

```text
Eres un experto en la clasificación de consultas. Tu tarea es determinar a qué tema pertenece la pregunta. Los temas posibles son: automóviles, comida, tecnología.
```

**Prompt del usuario**

```text
¿Cuál es su kilometraje en un coche eléctrico Tesla Model 3?
```

**Respuesta del clasificador LLM**

```json
{
  "predictions": [
    {
      "label": "automóviles",
      "confidence": 0.85
    },
    {
      "label": "comida",
      "confidence": 0.05
    },
    {
      "label": "tecnología",
      "confidence": 0.10
    }
  ]
}
```

