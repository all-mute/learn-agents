# Aplicaciones de LLM

Veamos para qué otras tareas se pueden aplicar los LLM en modo one-shot.

## Preguntas

- ¿Qué es NER - Reconocimiento de Entidades Nombradas?
- ¿Qué más, además de las famosas *resúmenes, traducciones, generación de código*, podemos pedirle a un LLM?
