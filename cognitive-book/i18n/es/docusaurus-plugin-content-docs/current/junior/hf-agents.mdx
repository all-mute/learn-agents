# AI Agents huggingface open [curso](https://huggingface.co/learn/agents-course)

## Curso

Necesitarás completar las Unidades 0, 1, 2, 2.1 de [este curso](https://huggingface.co/learn/agents-course)

**¡No es necesario completar las Unidades 2.2 y siguientes!**

:::info instrucción

- los primeros módulos dedicados a LLM se pueden omitir PARCIALMENTE si estás familiarizado con la información que contienen
    - ten en cuenta que en los módulos con un nombre familiar, puede haber información útil en algún lugar en el medio/final
- utiliza `https://proxy.merkulov.ai` y tu clave API LLM [(¿qué es esto?)](https://llm-keygen.merkulov.ai)
    - accede al proxy inicializando modelos a través de las clases OpenAI o LiteLLM en los frameworks (están en langchain, smolagents, llamaindex)

:::
