### Классификация вопроса по теме

**Системный промпт**

```text
Ты - эксперт по классификации запросов. Твоя задача - определить, к какой теме относится вопрос. Возможные темы: автомобили, еда, технологии.
```


**Пользовательский промпт**

```text
Какой у вас пробег на электромобиле Tesla Model 3?
```


**Ответ LLM классификатора**

```json
{
  "predictions": [
    {
      "label": "автомобили",
      "confidence": 0.85
    },
    {
      "label": "еда",
      "confidence": 0.05
    },
    {
      "label": "технологии",
      "confidence": 0.10
    }
  ]
}
```

### Классификация в RAG (относимость текста к вопросу)

**Системный промпт**

```text
Ты - эксперт по оценке релевантности текстов. Определи, относится ли данный чанк текста к заданному вопросу.
```


**Пользовательский промпт**

```text
Вопрос: Каковы преимущества гибридных автомобилей?
Текст: Гибридные автомобили сочетают в себе двигатель внутреннего сгорания и электродвигатель, что позволяет экономить топливо.
```


**Ответ LLM классификатора**

```json
{
  "predictions": [
    {
      "label": "релевантный",
      "confidence": 0.92
    },
    {
      "label": "нерелевантный",
      "confidence": 0.08
    }
  ]
}
```
