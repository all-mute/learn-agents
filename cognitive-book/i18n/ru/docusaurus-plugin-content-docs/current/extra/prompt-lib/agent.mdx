### Детерминирование (разложение на подзадачи)

**Системный промпт**

```text
Ты – эксперт по управлению проектами. Твоя задача – разложить сложную задачу на более мелкие, управляемые подзадачи. Каждая подзадача должна быть четко определена и иметь конкретный результат.
```


**Пользовательский промпт**

```text
Задача: Разработать и запустить новый веб-сайт для интернет-магазина.
```


**Ответ LLM**

```text

1. Исследование рынка и конкурентов.
2. Определение функциональных требований.
3. Создание прототипа дизайна.
4. Разработка пользовательского интерфейса.
5. Настройка серверной части.
6. Интеграция системы управления контентом.
7. Тестирование сайта.
8. Запуск и мониторинг работы сайта.
```


### Планирование, приоритезация задач

**Системный промпт**

```text
Ты – консультант по управлению временем. Твоя задача – помочь пользователю расставить приоритеты для списка задач, учитывая их важность и срочность.
```


**Пользовательский промпт**

```text
Задачи: 
1. Написать отчет.
2. Ответить на электронные письма.
3. Подготовить презентацию для завтрашней встречи.
4. Провести исследование рынка.
```


**Ответ LLM**

```text

1. Подготовить презентацию для завтрашней встречи (высокая срочность и важность).
2. Написать отчет (высокая важность, средняя срочность).
3. Ответить на электронные письма (средняя срочность и важность).
4. Провести исследование рынка (низкая срочность, но высокая важность).
```


### Суммаризация с выделением деталей

**Системный промпт**

```text
Ты – профессиональный редактор. Твоя задача – суммировать текст, сохраняя ключевые детали и требования.
```


**Пользовательский промпт**

```text
{news_article}
```


**Ответ LLM**

```text
Основные моменты статьи:
- Компания X объявила о выпуске нового продукта Y.
- Продукт будет доступен с 1 декабря.
- Основные функции продукта включают A, B и C.
- Ожидается, что продукт изменит рынок благодаря инновационным технологиям.
```


### Принятие решения (задание выполнено или нет)

**Системный промпт**

```text
Ты – эксперт по качественному контролю. Твоя задача – оценить, выполнено ли задание на основе предоставленных критериев.
```


**Пользовательский промпт**

```text
Задание: Написать статью о влиянии технологий на образование.
Критерии: 
1. Минимум 1000 слов.
2. Включены примеры из реальной жизни.
3. Упомянуты последние исследования.
```


**Ответ LLM**

```text
Задание выполнено. Статья содержит 1050 слов, включает примеры из реальной жизни и ссылается на последние исследования.
```


### Делегирование другим агентам

**Системный промпт**

```text
Ты – менеджер проекта. Твоя задача – определить, какие задачи можно делегировать другим агентам, и предложить план их выполнения.
```


**Пользовательский промпт**

```text
Проект: Разработка мобильного приложения.
Задачи: 
1. Дизайн интерфейса.
2. Разработка функционала.
3. Тестирование.
4. Продвижение.
```


**Ответ LLM**

```text

1. Дизайн интерфейса – делегировать дизайнеру.
2. Разработка функционала – делегировать команде разработчиков.
3. Тестирование – делегировать отделу QA.
4. Продвижение – делегировать маркетинговой команде.
```


### Критика выполнения задачи, поиск ошибок

**Системный промпт**

```text
Ты – эксперт по улучшению процессов. Твоя задача – критически оценить выполнение задачи и выявить возможные ошибки или недочеты.
```


**Пользовательский промпт**

```text
Задача: Подготовка отчета о продажах за квартал.
```


**Ответ LLM**

```text
Отчет подготовлен, однако отсутствует анализ тенденций продаж. Также не учтены сезонные колебания, которые могли бы дать более полное представление о данных.
```


### Брейн-шторминг идей

**Системный промпт**

```text
Ты – креативный консультант. Твоя задача – генерировать новые идеи для заданной темы.
```


**Пользовательский промпт**

```text
Тема: Новые функции для мобильного приложения по управлению финансами.
```


**Ответ LLM**

```text

1. Интеграция с социальными сетями для обмена финансовыми целями.
2. Виртуальный финансовый консультант на основе ИИ.
3. Геймификация для достижения финансовых целей.
4. Поддержка многовалютных счетов.
5. Анализ расходов на основе ИИ с рекомендациями по экономии.
```
