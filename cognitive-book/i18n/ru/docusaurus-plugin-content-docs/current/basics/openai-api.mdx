# Изучаем OpenAI API

## Steps

### 1. Читаем документацию OpenAI API

1. https://platform.openai.com/docs/api-reference/chat/create

2. Читаем всё про ручку `chat/completions`, ничего не пропускаем. Весь будущий материал будет строиться на этой ручке и вам важно в деталях понимать, как она работает, иначе вы будете путаться.

3. По функциям, читаем https://www.ai-cookbook.ru/docs/guides/tools_yc/FC_why целиком, 3 главы. *Не обращайте внимание на то, что там написано про Yandex Cloud, код работает с API формата OpenAI.*


### 2. Читаем странички из Cookbook

1. Как считать токены в тексте?
https://cookbook.openai.com/examples/how_to_count_tokens_with_tiktoken

2. Как выглядят чанки в потоковом ответе опенаи? Какая информация передается в первом, промежуточных и последнем чанке?
https://cookbook.openai.com/examples/how_to_stream_completions

3. По функциям, читаем https://www.ai-cookbook.ru/docs/guides/tools_yc/FC_why целиком, 3 главы. *Не обращайте внимание на то, что там написано про YandexGPT, код работает с API формата OpenAI.*
Аналог - https://cookbook.openai.com/examples/how_to_call_functions_with_chat_models

4. https://cookbook.openai.com/examples/how_to_use_structured_outputs
