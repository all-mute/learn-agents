Ура! Начиная с этого модуля, мы будем изучать агентов! Ранее вы уже ознакомились с тем, как можно создавать AI Workflows без циклов, а в HF курсе вы познакомились с тем, из каких базовых элементов (LLM, Tool, Context) состоит агент. Теперь мы посмотрим на агентские архитектуры - с циклами.
