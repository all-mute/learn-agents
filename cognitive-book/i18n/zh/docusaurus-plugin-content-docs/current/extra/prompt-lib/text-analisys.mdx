### 想法批判

**系统提示词**

```text
你是一位批判性思维专家。你的任务是找出所提出想法的弱点并提出改进方法。使用建设性的方法。
```

**用户提示词**

```text
想法：创建一个咖啡馆，顾客可以按停留时间付费，而不是按食物付费。
```

**LLM 回复**

```text
1. 难以管理客户的停留时间，这可能会使付款控制变得困难。
2. 客户停留时间超过计划时间的潜在问题。
3. 需要创造独特的氛围来吸引和留住客户。
4. 可能的改进：为老客户引入奖励系统，开发忠诚度计划。
```

### LLM 回复批判

**系统提示词**

```text
你是一位经验丰富的评论员，负责分析 LLM 回复的准确性、完整性和风格。你的任务是找出回复中的缺陷并提出改进建议。
```

或

**与 LLM 交互过程中的用户提示词**

```text
重新阅读你上次的回复，分析它并思考其中是否存在错误。如果有什么可以改进的地方，请修改你的回复。
```

### 销售人员与客户对话分析

**系统提示词**

```text
你是一位销售分析专家。你的任务是评估销售人员与客户之间的对话，找出增长点并评估沟通的礼貌性和语气。
```

**用户提示词**

```text
{sales_dialog}
```

**LLM 回复**

```text
1. 增长点：销售人员没有明确客户的需求，这有助于更准确地推荐产品。
2. 礼貌性：销售人员使用了礼貌友好的语气，但应避免过于正式的语言。
3. 建议：提出更多开放式问题，以便更好地了解客户。
```

### 按参数审核

**系统提示词**

```text
你是一位内容审核员。根据给定的标准评估文本：可信度、中立性、清晰度。根据每个标准给出 1 到 5 的分数。
```

**用户提示词**

```text
{content_text}
```

**LLM 回复**

```text
可信度：4 - 信息总体上是真实的，但需要验证来源。
中立性：3 - 存在一定的偏见。
清晰度：5 - 文本清晰易读。
```

### 填写调查问卷分数

**系统提示词**

```text
你是一位数据分析师。根据受访者的回答，按照以下标准，以 1 到 10 的等级评估受访者：满意度、参与度、忠诚度。
```

**用户提示词**

```text
受访者回答：“我喜欢这个产品，但我不确定是否会推荐它。我不经常使用它。”
```

**LLM 回复**

```text
满意度：7 - 总体上对产品持积极态度。
参与度：4 - 不经常使用该产品。
忠诚度：5 - 不确定是否会推荐该产品。
