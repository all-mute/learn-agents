### 文本的简短总结，用1个句子/段落

**系统提示词**

```text
你是一位文本总结专家。你的任务是将长文本简要地总结成一个句子或段落，保留主要思想。
```

**用户提示词**

```text
{long_text}
```

**LLM 回复**

```text
文本描述了公司过去一年的主要成就，包括利润增长和向新市场的扩张。
```

### 文本的简短总结，用3-5个句子

**系统提示词**

```text
你是一位总结专家。你的任务是用3-5个句子概括文本的要点，保留关键细节。
```

**用户提示词**

```text
{long_text}
```

**LLM 回复**

```text
公司实现了显著的利润增长，增加了其市场份额。在这一年中，它扩大了在亚洲和欧洲的业务，推出了几个新产品。这些成就得到了成功的营销活动和生产创新的支持。
```

### 带有细节的总结

**系统提示词**

```text
你是一位详细总结专家。复述文本，保留所有重要的细节和事实。
```

**用户提示词**

```text
{detailed_text}
```

**LLM 回复**

```text
报告指出，由于在亚洲的成功战略，公司利润增长了20%，并在那里开设了三个新办事处。针对年轻人的营销活动导致销售额增长了15%。此外，在生产中引入新技术使成本降低了10%。
```

### 将文本重写为 X 格式（正式、有趣、创意、其他语言）

**系统提示词**

```text
你是一位文本重写大师。以 {desired_style} 风格重写文本。
```

**用户提示词**

```text
文本: "今天我们与合作伙伴举行了一次成功的会议。"
风格: 有趣
```

**LLM 回复**

```text
今天我们与合作伙伴举行了一次精彩的会议，就像一场节日一样——每个人都带着微笑和新想法离开了！
```

### 带有细节的文本重写

**系统提示词**

```text
你是一位在文本中添加细节的专家。通过添加更多细节和上下文来扩展文本。
```

**用户提示词**

```text
文本: "我们参加了一个会议。"
```

**LLM 回复**

```text
我们参加了在旧金山举行的国际技术会议，在那里我们与行业领先的专家会面，并讨论了人工智能领域的最新趋势。
```

### 生成 SEO 文本

**系统提示词**

```text
你是一位 SEO 专家。编写针对搜索引擎优化的文本，使用关键词："智能家居"、"技术"、"创新"。
```

**用户提示词**

```text
主题: 智能家居的优势
```

**LLM 回复**

```text
智能家居是一个创新系统，它使用先进的技术来提高您住所的舒适性和安全性。通过集成现代解决方案，如语音控制和自动化，智能家居为节能和改善生活质量提供了独特的机会。了解更多关于智能家居技术领域的最新创新，并立即开启您的未来。
```

### 根据参数编写文本：评论、电子邮件、产品卡片、职位描述

**系统提示词**

```text
你是一位根据给定参数创建文本的专家。编写 {text_type}，同时考虑以下参数：{parameters}。
```

**用户提示词**

```text
文本类型: 评论
参数: 产品 - 智能手机，评分 - 5 星，特点 - 出色的相机，电池续航时间长。
```

**LLM 回复**

```text
我对这款智能手机非常满意！即使在低光照条件下，相机也能拍摄出令人惊叹的照片，而且电池即使在积极使用的情况下也能持续一整天。绝对推荐购买！
