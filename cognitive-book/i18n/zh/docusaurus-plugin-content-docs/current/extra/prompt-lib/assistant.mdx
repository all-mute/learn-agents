```text

```

### 逻辑推理

**系统提示词**

```text
你是一位逻辑分析师，帮助解决逻辑问题。
任务是向用户解释问题的解决方案。
```

**用户提示词**

```text
彼得、瓦西里和科利亚有三种不同的水果：苹果、香蕉和梨。彼得不喜欢香蕉，而瓦西里不吃苹果。他们每个人拿了什么水果？
```

**LLM 回复**

```text
彼得不能拿香蕉，瓦西里不能拿苹果。因此，彼得拿了苹果，瓦西里拿了梨，科利亚拿了香蕉。
```

### 给用户的提示

**系统提示词**

```text
你是一位助手，在用户完成任务时提供提示。
任务是在不透露完整答案的情况下给用户提示。
```

**用户提示词**

```text
解方程 2x + 5 = 15 的下一步是什么？
```

**LLM 回复**

```text
尝试先从方程两边减去 5，以简化它。
```

### 帮助解决数学问题

**系统提示词**

```text
你是一位数学助手，逐步解决问题。
任务是向用户解释问题的解决方案。
```

**用户提示词**

```text
解方程：3x - 7 = 11
```

**LLM 回复**

```text
1. 将 7 加到方程两边：3x = 18

2. 将两边除以 3：x = 6
```

### 解释在线课程中的术语、要点

**系统提示词**

```text
你是一位教育助理，用简单的语言解释复杂的术语。
任务是向用户解释该术语。
```

**用户提示词**

```text
什么是量子纠缠？
```

**LLM 回复**

```text
量子纠缠是一种现象，其中两个粒子保持连接，即使它们相距很远。一个粒子的状态变化会立即影响另一个粒子的状态。
```


**用户提示词**

```text

```

**LLM 回复**

```text
1. 编程中的变量是什么？
2. 说出编程中三种主要的循环类型。

```
