# AI 代理：技术新纪元

人工智能代理正在迅速改变技术格局，为开发者和研究人员开启前所未有的机遇。
我们为什么要研究这个领域？

## 1. 代理非常棒！

- **2022**: [StreamLit-PandasAI](https://github.com/worldline/streamlit-pandasai) - 将人工智能集成到分析工具中
- **2023**: [You.com](https://you.com/) 处于自主研究模式 - 一种基于代理的搜索引擎的新方法
- **2023**: 出现自主的 Chat with data 系统 - "使用我们的数据仓库构建过去 5 年用户 LTV/CAC 变化仪表板"
- **2024**: [Junior ML Engineer](https://youtu.be/q1XFm21I-VQ?si=70kO1tBiccm6J47T&t=994) (观看 3 分钟，16:30 - 19:00) by [Andrew Ng](https://www.andrewng.org/)
- **2025**: [Manus](https://youtu.be/K27diMbCsuw?si=vhvBBzXLRG81ZUPz) (4 分钟视频)

## 2. 资金涌入！

2024 年，人工智能代理市场估值为 3000 亿美元，而目前的消费仅为 16 亿美元。这表明了巨大的增长潜力。

#### 大规模投资：
- **美国**: [$5000 亿](https://openai.com/index/announcing-the-stargate-project/)
- **欧洲**: [$1000 亿](https://www.reuters.com/technology/artificial-intelligence/france-invest-109-billion-euros-ai-macron-announces-2025-02-09/)
- **中国**: $3000 亿

所有这些资金都用于发展工业人工智能，从而对合格的专家产生了巨大的需求。

#### 专家短缺

关于人工智能代理的第一批科学出版物仅在 2023 年夏季出现。2024 年，第一批课程、应用程序和基础设施开始出现。世界上大多数教师还没有时间了解这个领域，而少数深入研究这个主题的学生已经创立了自己的初创公司并自己雇用员工。

#### 扩展问题：

如今，人工智能初创公司面临的主要问题之一是人工智能代理工程师团队的横向扩展。公司急于占据其细分市场的主导地位，但市场上几乎没有具备基本知识的专家。

#### 职业前景

[AI Agents - исследование зарплат для RU/EU/US рынков](https://www.reddit.com/r/AI_Agents/comments/1jhtu6v/genai_frameworks_popularity_on_job_market_research/?utm_source=share&utm_medium=web3x&utm_name=web3xcss&utm_term=1&utm_content=share_button)

- **欧洲的薪水**起价为每年 60,000 欧元
- **美国的薪水**起价为每年 100,000 美元
- **Web3 + AI** 提供更高的薪酬

要开始在该领域的职业生涯，只需创建 3 个小型项目即可。

#### 对专家的需求量很大

每月都有数百家开发人工智能代理的初创公司获得投资。科技巨头内部正在启动新的方向。公司无法承担花费半年时间寻找高级专家的费用，因此更喜欢聘用合适的候选人并培训必要的技能。

## 结论：

- **行业投资**：巨大
- **对工程师的需求**：极高
- **专家数量**：极少
- **教育基础设施**：正在形成
