# AI 智能体正在改变世界

在 2025 年，世界正被 AI 智能体席卷，但它们还不能自我创建。即使是像 AWS 这样的科技巨头也承认智能体平台的巨大潜力。不仅是大型科技公司，一大批初创公司现在也放弃了传统的聊天机器人，转而使用更先进的基于智能体的系统。

虽然其中一些平台除了登陆页面上的文字外什么都没改变，但很明显，AI 智能体开发人员的角色将成为 2025 年最受欢迎的职业之一。然而，由于这个方向仍处于早期阶段，没有人真正理解该学什么，更不用说如何成为这样的开发人员了。

在本文中，我将最终揭示 AI 智能体开发人员是谁，你需要哪些技能，最重要的是，提供一个完整的路线图，你可以立即开始遵循。让我们开始吧！

## 什么是 AI 智能体

现在让我们弄清楚什么是 AI 智能体。大多数科技影响者将其定义为自动化或 AI 员工。这两种定义都会导致误解。

一方面，AI 智能体比普通自动化具有更大的自主性。无论如何，AI 自动化都在你的后端运行——即使整个世界都烧毁了，自动化也会像什么都没发生一样继续运行。

另一方面，AI 智能体比组织自己的员工具有更少的自主性，因为员工可以不服从管理。你可以给他们任意多的指示，但他们仍然可以做出自己的决定并按照自己的方式行事。

最准确的定义是：智能体是自主执行者，他们计划、执行并完成任务的解决方案。

目前，阻碍 AI 智能体大规模采用的最大问题是它们的可靠性。为了使智能体可靠，需要根据特定的业务流程仔细配置它们，而这正是 AI 智能体开发人员发挥作用的地方。

## AI 智能体开发人员的角色

作为 AI 智能体开发人员，你的角色是确保你的智能体可以访问执行任务所需的所有必要资源、工具和知识。

本质上，这包括：

1. 进行彻底的业务研究，并确定智能体需要哪些资源
2. 创建工具（正如我已经说过的，它们是创建智能体时最重要的）
3. 不断迭代，收集利益相关者的反馈，并在循环中进行调整，直到智能体稳定运行
4. 部署智能体并将其集成到客户的业务中，以便它们与员工顺利协作

## 必要的技能

成为 AI 智能体开发人员需要哪些技能？

至于软技能——沟通能力和不断学习的意愿。沟通能力很重要，因为开发人员经常需要从利益相关者那里收集需求并建立明确的期望，这是一项具有挑战性的任务。不断学习的意愿是必要的，因为 AI 行业处于早期发展阶段，需要快速适应。

至于硬技能——主要是轻量级的后端开发与 AI 相结合。

## 2024 年每个人都可以成为 AI 智能体开发人员吗？

在我看来，是的，每个人都可以做到。想象一下，你试图在一个像数学这样的科目中取得成功——你将不得不与那些一生都致力于此的人竞争。然而，当涉及到创建 AI 智能体时，你只会与那些在该领域只有几个月经验的人竞争。

所以现在是理想的时机，即使你是新手。

## 是否需要会编程，还是使用 no-code 平台就足够了？

简短的回答：是的，仍然需要代码。这有几个原因：

1. 使用最强大的工具需要编程经验。即使在连接到现成的 API 时，在实际环境中通常也会创建代理服务器，以便更好地控制工具的行为。

2. 此外，许多工具需要组合多个 API 或添加自己的逻辑。

3. 开源将始终处于前沿——不仅在创建最强大的模型方面，而且在使用它们进行创造性活动方面。

4. 许多公司非常关注数据隐私。对于他们来说，SaaS 平台甚至不是一个选择——他们希望将智能体部署在他们现有的基础设施中。

虽然 SaaS 平台无疑会发展，但专家认为，掌握代码仍然是必要的。

此外，随着 Cursor 的出现，编写代码的过程变得类似于使用另一个 SaaS 平台，因为现在可以使用自然语言编写代码。
