# LLM 的适用性

让我们看看 LLM 还可以应用于哪些 one-shot 模式的任务。

## 问题

- 什么是 NER - 命名实体识别？
- 除了众所周知的*总结、翻译、代码生成*之外，我们还可以向 LLM 请求什么？
- LLM 是否可以扮演管理者的角色 - 编写提示词、向 Google 发出请求、为开发人员编写技术规范？

## 步骤

### 1. 学习提示词（没有其他快速学习的方法）

访问以下两个网站：

1. https://www.promptingguide.ai/prompts

2. [llm use cases in extra section](../extra/prompt-lib/)

跳过那些你从名称上 **1000%** 了解的用例。 最好阅读所有其他内容，以提高你的眼界和提示词-*直觉*。
