# 关于本模块

## Basics 模块的内容

:::info
在 Junior 模块中，我们将深入实践并开始掌握工业工具。到最后，你将能够创建工业级的代理：你将拥有实践经验、项目组合并了解 AgentOps。

在本模块结束时，你就可以开始在应用人工智能市场上寻找中级和入门级职位。
:::

### 开始

在本模块的开始，你将掌握 smolagents 教学框架，并通过包含实践项目的课程。

### RAG

在 RAG 模块中，你将掌握自然语言处理 (NLP) 数据预处理、RAG、向量数据库、嵌入。

### Frameworks grinding

在 Frameworks grinding 模块中，你将选择一个工业框架，并为你日常生活实现一个大型工业项目。

### Infrastructure

在 Infrastructure 模块中，你将获得对当前 AgentOps 格局的了解。
