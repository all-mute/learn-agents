# 云和虚拟机

在这个阶段，我们将掌握云技术和 VPS 的使用——这是你的 AI 代理赖以生存的数字基础。作为虚拟生物的架构师，你需要了解它们的“家”的结构：如何快速扩展不断增长的模型的居住空间，为不同的服务组织安全的“房间”，以及在组件之间铺设“通信”。这些技能将成为你部署模型、处理数百万个请求和创建容错系统的超能力。
