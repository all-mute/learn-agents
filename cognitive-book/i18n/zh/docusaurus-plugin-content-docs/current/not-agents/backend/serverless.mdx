# 快速开发

想象一下，你是一位未来的建筑师，用数字乐高积木建造摩天大楼。在这里，我们将掌握无服务器架构——你用于即时构建 AI 系统的神奇构造器。你将学会像魔术师从帽子里放出鸽子一样在云端运行神经网络 (AWS Lambda)，在几分钟内构建智能 API (Firebase)，并创建可随你的雄心壮志一起扩展的可扩展系统。这些技能将成为你进入工业 GenAI 开发领域的个人传送器，在这里，基础设施只需点击几下即可管理，资源就像宇宙一样无限。

:::tip Y Combinator 创业学校建议：
"Always use off-the-shelf solutions when possible."

（尽可能使用现成的解决方案。）
:::
